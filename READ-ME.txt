How to run the build

Open a terminal or cmmd at the build folder ( ./build/web-mobile )

Install any web server like system.

For this guid, I use Python.

For MacOS run it on terminal:

python3 -m http.server 8000 ( or any other unsed port )
Access http://localhost:8000 and enjoy the game

For windows:

python -m htp.server 8000

For Linux:
py3 -m http.server 8000}
